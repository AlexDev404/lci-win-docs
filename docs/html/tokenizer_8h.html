Detailed Description

Structures and functions for grouping lexemes into tokens. The tokenizer reads through an array of lexemes (generated by the lexer) and groups them into tokens based on their structure. In addition, some lexemes with semantic meaning (such as integers, floats, strings, and booleans) will have their values extracted and stored.

◆ isKeyword()

Token* isKeyword ( LexemeList *  lexemes,
unsigned int *  start 
)

Checks if the next lexemes in a list comprise a keyword and, if so, generates a new token representing that keyword. Specifically, lexemes is searched, starting at start for keywords. If one is found, an appropriate token is created and returned and start is incremented by the number of lexemes matched minus one.

Parameters
lexemes[in] A list of lexemes to search for keywords in.
start[in,out] The position within lexemes to begin searching for keywords.
Postcondition
If a keyword is not found, start will not be modified. Otherwise, start will be incremented by the number of lexemes matched minus one.
Returns
A pointer to the token containing the matched keyword.
Return values
NULLNo keywords were found or there was an error allocating memory.
