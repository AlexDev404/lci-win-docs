enum ErrorType

Represents an error type. The error types are organized based on which module they occur in:

  • MN_* for the main body,
  • LX_* for the lexer,
  • TK_* for the tokenizer,
  • PR_* for the parser,
  • IN_* for the interpreter
Note
Remember to update the error message and error code arrays (in the error C file) with the appropriate error message and code.
